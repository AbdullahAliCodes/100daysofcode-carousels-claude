Meet
localStorage.

Your browser's filing cabinet. Key-value store. Survives refresh. Survives tabs closing. Built into every modern browser. Free.

[API] setItem() · getItem() · removeItem() · clear()
[TYPE] strings only. JSON converts anything.
[LIMIT] ~5MB per origin. Plenty for notes.

Saving is
one function.

app.js
// write the notes array to disk saveNotes() { localStorage.setItem("notes", JSON.stringify(this.notes)); } // every CRUD op calls render() — not displayNotes() anymore render() { this.saveNotes(); this.displayNotes(); }
[ INSIGHT ] One render() wraps two jobs: persist, then repaint. Call it from every CRUD method and you literally cannot forget to save.

[ CODE ] 02 · LOAD ON STARTUP

Loading is
one line.

app.js · constructor()
class App { constructor() { // rehydrate notes from disk on boot this.notes = JSON.parse( localStorage.getItem("notes") ) || []; this.addEventListeners(); this.displayNotes(); } }
[ GOTCHA ] The || [] fallback is the unsung hero. First ever load has no "notes" key — without it, JSON.parse(null) returns null and .map() crashes with TypeError.
