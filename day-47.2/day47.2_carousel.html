NAVIGATION
Link vs NavLink
// NavLink — adds "active" class when URL matches // use in nav menus so you can style active state <NavLink to="/home">Home</NavLink> <NavLink to="/products">Products</NavLink> // Link — plain navigation, no active state // use for links within page content <Link to="/products/01">Product 1</Link> <Link to="/products/02">Product 2</Link>
NavLink
navigation menusauto-applies active class — easy to style the current page
Link
links inside contentno active class needed — just navigates cleanly
